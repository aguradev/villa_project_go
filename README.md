# Reservation Villa Go - API

## About

Villa Reservation REST API System is a platform that allows users to search, view information and reserve villas for their holidays or special events. The system utilizes a RESTful API architecture to enable communication between clients (such as mobile applications or websites) and backend servers providing villa reservation services.

## Tech Stack

- [Echo](https://echo.labstack.com/): Web Framework
- [Gorm](https://gorm.io/): Golang ORM
- [Postgresql](https://www.postgresql.org/): Relational Database
- [Docker](https://www.docker.com/): Containerization
- [GCP](https://cloud.google.com//): Cloud Platform Deployment
- [Cloudflare R2](https://www.cloudflare.com/developer-platform/r2/): Cloud Storage
- [Midtrans](https://midtrans.com): Payment Gateway
